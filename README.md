## 🎨 設計特色

### 視覺標記
- **🟡 待填寫** - 黃色標記，表示需要你填入資料
- **✅ 已完成** - 綠色勾選，表示內容完整
- **⚠️ 重要提醒** - 警告框，強調重點
- **💡 補充說明** - 資訊框，提供額外資訊

### 顏色系統
- **深藍** (#2c3e50) - 主要標題
- **青綠** (#27ACB2) - 強調重點（ONE桌遊品牌色）
- **黃色** (#fff3cd) - 待填寫標記
- **淺灰** (#f8f9fa) - 背景色

1. **編輯 index.html**
   ```bash

   # 用編輯器打開 index.html
   # 搜尋 "待填寫" 並填入資料
   ```

2. **提交並推送**
   ```bash
   git add index.html
   git commit -m "更新：填入薪資福利資訊"
   git push
   ```

3. **等待部署**（1-2 分鐘）
   - GitHub Pages 會自動重新部署
   - 刷新網頁即可看到更新

### 更新 PowerPoint

1. **重新生成**（如果修改了生成腳本）

   ```bash
   cd ~/.openclaw/workspace/company-training
   python3 generate_ppt.py
   ```

   - 直接編輯內容
   - 儲存即可

## 💡 使用建議

### 新人報到流程
1. **報到前 3 天**：發送網頁版連結給新人預先閱讀
2. **Day 1 下午**：簡報講解（1.5-2 小時）
3. **重點投影片**：
   - ⭐⭐⭐ 投影片 8-9（工作期待）
   - ⭐⭐ 投影片 11-13（薪資、第一週）
   - ⭐⭐ 投影片 16（緊急處理）
4. **互動討論**：每 5-6 張暫停提問
5. **會後跟進**：發送 PowerPoint 供新人複習

### 講解時間分配
- 第一部分（1-4）：10 分鐘（快速帶過）
- 第二部分（5-7）：15 分鐘（重點強調）
- **第三部分（8-10）：30 分鐘**（最重要！）
- 第四部分（11-21）：40 分鐘（逐項說明）
- 第五部分（22-23）：10 分鐘
- 第六部分（24-25）：5 分鐘
- **總計**：約 110 分鐘（1小時50分）
