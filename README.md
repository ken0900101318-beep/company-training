# 遊戲家資訊科技 - 新人教育訓練簡報

## 📦 檔案清單

```
company-training/
├── index.html                    # 網頁版簡報（Reveal.js）
├── 遊戲家新人教育訓練.pptx        # PowerPoint 版（可下載編輯）
├── generate_ppt.py               # PowerPoint 生成腳本
└── README.md                     # 本說明文件
```

---

## 🌐 網頁版使用方式

### 開啟簡報
```bash
# 在瀏覽器開啟
open index.html

# 或使用本地伺服器（推薦）
python3 -m http.server 8000
# 然後開啟 http://localhost:8000/
```

### 鍵盤操作
- **→** / **Space** - 下一頁
- **←** - 上一頁
- **ESC** - 總覽模式（查看所有投影片）
- **F** - 全螢幕
- **S** - 演講者模式（顯示筆記）
- **B** / **.** - 黑屏（暫停）

### 分享連結
將 `index.html` 上傳到 GitHub Pages 或任何網頁伺服器即可分享。

---

## 💼 PowerPoint 版使用方式

### 開啟簡報
直接雙擊 `遊戲家新人教育訓練.pptx` 即可。

### 編輯內容
1. 在 PowerPoint / Keynote / Google Slides 中開啟
2. 編輯文字、顏色、版面
3. 儲存後即可使用

### 重新生成
如需修改程式生成的內容：
```bash
# 編輯 generate_ppt.py
vim generate_ppt.py

# 重新生成
python3 generate_ppt.py
```
